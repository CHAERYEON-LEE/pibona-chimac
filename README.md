## 🍗 피보나치킨 클론 로직 

### 1. 피보나치수열 정의

현재 항의 값(n)은 이전 값의 항(n-1)과 이전 값의 전의 항(n-2)의 합을 피보나치 수열이라고 한다.

### 2. 피보나치 수열의 황금비

피보나치 수열의 황김비는 피보나치 수열 안이 주어졌을 때 바로전에 오는 수자의 비율이다.

### 3. 제켄도로프의 정의

다른 자연수에 대한 피보나치수열의 황금비를 피보나치수열의 합으로 나타낸다.
